Website owner checklist (recommended)

If your goal is: “I want visitors to zap me on my website”, follow these steps.

  1. Prepare your Nostr profile for zaps. Your profile metadata should include a Lightning address / LNURL (commonly lud16 or lud06). If it’s missing, zaps cannot generate an invoice.
  2. Include the script bundle (one line). See the “Embed” card below.
  3. Add a Zap button using data-npub (for profiles) or data-note-id / data-naddr (for content).
  4. Pick relays (recommended: multiple read+write relays). This demo uses a curated relay list.
  5. Pick a theme by setting data-theme to light or dark. Use the selector on this page to try both.
Tip: start with a profile button first. If profile zaps work, note zaps will usually work too.

Embed (1 line)

Add this script tag near the bottom of your HTML (before </body>). That’s it — the library auto-wires buttons by attributes.

Copy script tag 
<script src="https://zap.nostr.buzz/dist/nostr-zap.js"></script>

Minimal button snippet

Copy/paste this into your site and replace npub1... with your own identifier. You can style the buttons however you want.

Show HTML 
<div style="display:inline-flex;gap:8px">
  <button data-npub="npub1..."
    data-relays="wss://relay.damus.io,wss://relay.primal.net,wss://nos.lol">
    Zap
  </button>
  <button data-nzv-id="npub1..."
    data-zap-color-mode="true"
    data-relay-urls="wss://relay.damus.io,wss://relay.primal.net,wss://nos.lol">
    View
  </button>
</div>
